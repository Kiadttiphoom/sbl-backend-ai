# Search Data Skill
ใช้สำหรับดึงข้อมูลจากระบบ SBL เพื่อนำมาตอบคำถามหรือวิเคราะห์เบื้องต้น:
- ค้นหาพนักงานจากชื่อ หรือ รหัสพนักงาน (FolID)
- ค้นหาข้อมูลสัญญา (AccNo) และสถานะการติดตาม (Stat2)
- ดึงข้อมูลลูกหนี้ตามเงื่อนไข เช่น ลูกหนี้สูญ, ติดคดี, หรือ บอกเลิก 35 วัน
- ตรวจสอบยอดหนี้คงเหลือ ยอดภาษี และดอกเบี้ยของรายสัญญา
- **หมายเหตุ:** หากต้องการสรุปภาพรวมจำนวนมาก ให้ใช้ควบคู่กับ analyze-report หรือหากซับซ้อนเกินไปให้ใช้ AGENT_FALLBACK
